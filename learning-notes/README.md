# Learning Notes

这个目录用于保存按 `Learning Todo List` 实践过程中的学习笔记、阶段产出和复习材料。

## Start Here

下次重开上下文时，先读 [Learning Progress](PROGRESS.md)。它只记录当前进度、续接锚点和下一步动作，用来快速开始，不替代阶段笔记。

## 目录

- [Learning Progress](PROGRESS.md)
- [Stage 0: Agent 基础概念](stage-0-agent-basics.md)
- [阶段笔记模板](templates/stage-note-template.md)

## 文件分工

- `PROGRESS.md`：当前学习位置、下次从哪里开始、待回答检查题、完成规则。
- `stage-*-*.md`：每个阶段的完整学习内容、回答、修正、验证和阶段产出。
- `templates/`：后续阶段复用的笔记模板。

## 记录原则

- 每完成一个学习任务后，整理核心概念、自己的回答、修正点和下一步任务。
- 区分“已经理解/完成”和“待阅读/待实践”的内容。
- 笔记优先服务复习，不替代原 README 的公开路线说明。
- 只有在回答或实践经过检查、修正并写入阶段笔记后，才把 checklist 项标记为完成。

## 更新流程

1. 先更新对应阶段笔记，保留概念、回答、修正点和验证方式。
2. 再更新 `PROGRESS.md`，写清楚当前状态、续接锚点、待办问题和下一步。
3. 只有新增笔记、模板或学习产物时，才更新本索引。
